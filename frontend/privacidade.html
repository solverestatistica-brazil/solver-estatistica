Privacidade

Política de Privacidade

Última atualização: 15 de julho de 2026.

Dados processados

O Solver recebe os dados experimentais enviados pelo usuário e as configurações necessárias para executar a análise solicitada.

Finalidade e retenção

Os dados são processados para calcular resultados e gerar exportações. A versão pública não cria conta, histórico ou banco de projetos e não armazena intencionalmente os dados experimentais após o término da requisição.

Registros operacionais

O provedor de infraestrutura pode manter registros técnicos temporários, como horário, endereço IP, rota acessada, duração, código de resposta e identificador de erro. O aplicativo não deve registrar o conteúdo das planilhas ou payloads.

Compartilhamento

O processamento utiliza provedores de hospedagem e distribuição indispensáveis ao funcionamento do serviço. Os dados não são vendidos nem usados para treinamento de modelos.

Direitos do titular

Solicitações relacionadas à LGPD podem ser abertas no repositório oficial. Não inclua dados pessoais ou planilhas no chamado público.

Segurança

Apesar das medidas técnicas adotadas, nenhum serviço conectado à internet é isento de risco. Não envie dados pessoais ou segredos comerciais desnecessários à análise.

Revisão necessária: este texto deve ser validado pelo responsável legal antes da versão 1.0.
